HTML Colors


HTML colors allow us To using predefined color names, or RGB, HEX, HSL, RGBA, HSLA values.

Such As :

Color Name
Red
Orange
Yello
Cyan

RGB (Red, Green, Blue)

The RGB value defines HTML color by mixing red, green, and blue values. The first number describes the red color input, the second – the green color input, and the third one – the blue color input.

Each parameter (red, green, and blue) defines the intensity of the color between 0 and 255.

Color Name
rgb(255,0,0)
rgb(255,255,0)
rgb(0,255,0)
rgb(0,255,255)
rgb(0,0,255)

Shades of gray are often defined using equal values for all the 3 light sources:

Color Name
rgb(255,0,0)
rgb(255,255,0)
rgb(0,255,0)

Hexadecimal Colors

With HTML, RGB values can also be specified using hexadecimal color values in the form: #RRGGBB, where RR (red), GG (green) and BB (blue) are hexadecimal values between 00 and FF (same as decimal 0-255).

For example, #FF0000 is displayed as red, because red is set to its highest value (FF) and the others are set to the lowest value (00).

Color Name
#FF0000
#FF0000
#00FF00
#00FFFF
#0000FF

Shades of gray are often defined using equal values for all the 3 light sources:

Color Name
#000000
#808080
#FFFFFF
